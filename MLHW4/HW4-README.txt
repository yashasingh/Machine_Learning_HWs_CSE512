Running the code instructions:


Structure :        python3 filename.py --dataset [data path] --K [value] --distance [distance metric]


For example: 
(a) Euclidean distance as the metric 
1. python3 kmeans.py --dataset Breast_cancer_data.csv --K 2 --distance euclidean


(b) Manhattan distance as the metric 
1. python3 kmeans.py --dataset Breast_cancer_data.csv --K 2 --distance manhattan
